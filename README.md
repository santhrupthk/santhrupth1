## Viral Structure

### Hook (0-2 sec)
Start with a dramatic transformation.

### Build-Up (3-10 sec)
Rapid visible progress and scale increase.

### Payoff (10-20 sec)
Reveal the final cinematic result.

### Loop
End frame connects smoothly to beginning.
